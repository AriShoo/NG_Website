Game Information

Hello! Welcome to the home of "Lily Pad Hop".
This game is created by Ari Shoo Designs

Lily Pad Hop is a game where you play as a frog trying to start a family.

The goal of the game is to get the frog to it's mate by crossing various obstacles and avoiding various enemies.

Once you reach your mate you're ready to play the next level, but as a baby of the previous frog!

This is a project that has been in the making as part of the college course HND: Digital Design and Development (Games). More information can be found about the course by following the hyperlink.
